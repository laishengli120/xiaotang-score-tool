# 班级各科评分表生成器

一个用于小六成绩 Excel 的静态网页工具。文件在浏览器本地读取和计算，生成包含以下工作表的新 Excel：

- 原始成绩
- 计算用成绩
- 三率计分表
- 班级各科评分表
- 规则说明

## 计算规则

- 先按目标县区过滤，默认 `12小塘镇`；旧表里的 `小塘镇` 也会被匹配。
- 班级代码会按文本归一，例如数字 `61` 和文本 `61` 归为同一班。
- 得分率 = 科目平均分 / 科目满分。
- 合格率 = 达到合格线人数 / 参考人数，默认合格线为满分的 `60%`。
- 优秀率 = 达到优秀线人数 / 参考人数，默认优秀线为满分的 `85%`。
- 各科计分 = 得分率归一计分 + 合格率归一计分 + 优秀率归一计分。
- 默认权重为得分率 40、合格率 40、优秀率 20。
- 最终各科得分 = 各科三率计分合计 × 办学系数。

## 本地运行

```bash
python3 -m http.server 4173
```

打开 `http://127.0.0.1:4173`。
